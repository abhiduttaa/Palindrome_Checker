# 🔁 Palindrome Checker

Check whether a word or phrase is a palindrome — clean, simple, and beautiful.  
This web app is built using HTML, CSS, and JavaScript, with a modern linear gradient UI and responsive layout.

✨ **Live Demo**: [abhiduttaa.github.io/Palindrome_Checker](https://abhiduttaa.github.io/Palindrome_Checker/)

## 🚀 Features

- 🔤 Checks any text input for palindromes
- ⚡ Instant result with dynamic message
- 🎨 Clean UI with dark-themed linear gradient
- 📱 Fully responsive for mobile and desktop
- 💡 Highlights result with color coding (success/error)

---

## 🛠️ Tech Stack

- **HTML5**
- **CSS3**
- **JavaScript (Vanilla)**
